# Image Background Remover — MVP 需求文档

## 1. 产品概述

**产品名称：** Image Background Remover  
**定位：** 纯前端、免费、无需上传服务器的 AI 图片背景移除工具  
**核心卖点：** 隐私安全（图片不离开浏览器）、免费无限次使用、无需注册

## 2. 目标用户

- 需要快速抠图的设计师、运营人员
- 电商卖家（产品图去背景）
- 社交媒体内容创作者
- 对隐私敏感、不愿上传图片到第三方服务器的用户

## 3. 技术方案

### 3.1 核心技术

- **抠图引擎：** `@imgly/background-removal`（基于 ONNX Runtime Web，浏览器端运行 AI 模型）
- **前端框架：** Next.js 15 + TypeScript + Tailwind CSS
- **部署：** Cloudflare Pages（静态导出）

### 3.2 技术优势

- 所有计算在浏览器本地完成，图片不上传到任何服务器
- 基于 U2Net / ISNet 等成熟分割模型，效果接近商业方案
- WASM + WebGPU 加速，现代浏览器性能良好

## 4. MVP 功能清单

### P0 — 必须实现

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持点击上传 + 拖拽上传，支持 JPG/PNG/WebP |
| AI 背景移除 | 调用 @imgly/background-removal 在浏览器端处理 |
| 处理进度 | 显示模型加载进度 + 处理进度条 |
| 结果预览 | 左右对比（原图 vs 去背景），棋盘格背景显示透明区域 |
| 下载结果 | 一键下载 PNG（保留透明通道） |
| 响应式布局 | 移动端 + 桌面端自适应 |

### P1 — 优先实现

| 功能 | 描述 |
|------|------|
| 自定义背景色 | 去背景后可选择填充纯色背景（白/黑/自定义颜色） |
| 批量处理 | 支持同时上传多张图片，队列处理 |
| 图片粘贴 | 支持 Ctrl+V 从剪贴板粘贴图片 |

### P2 — 后续迭代

| 功能 | 描述 |
|------|------|
| 自定义背景图 | 上传自定义背景图替换 |
| 边缘优化 | 手动调整边缘羽化、平滑度 |
| 历史记录 | 本地存储处理历史（IndexedDB） |
| PWA 支持 | 离线可用 |

## 5. 页面结构（MVP）

```
/ (首页 = 工具页，单页应用)
├── Header：Logo + 标题 + GitHub 链接
├── Hero：一句话介绍 + 隐私卖点提示
├── Upload Zone：拖拽/点击上传区域
├── Processing：进度条 + 状态提示
├── Result：原图/结果对比 + 下载按钮 + 背景色选择
└── Footer：技术说明 + 隐私声明
```

## 6. 用户流程

```
打开网站 → 上传/拖拽/粘贴图片 → 自动开始处理（首次需加载模型 ~30MB）
→ 显示进度条 → 处理完成 → 左右对比预览 → 下载 PNG
```

## 7. 非功能需求

| 项目 | 要求 |
|------|------|
| 首屏加载 | < 2s（不含模型下载） |
| 模型缓存 | 首次加载后缓存到浏览器，后续秒开 |
| 浏览器兼容 | Chrome 90+、Edge 90+、Safari 16+、Firefox 100+ |
| 最大图片 | 支持 4096x4096 以内 |
| SEO | 基本 meta 标签 + Open Graph |

## 8. 设计风格

- 简洁现代，参考 remove.bg 的简洁风格
- 主色调：深色/渐变紫蓝（科技感）
- 大量留白，聚焦核心操作
- 动效：上传动画、处理进度、结果淡入
